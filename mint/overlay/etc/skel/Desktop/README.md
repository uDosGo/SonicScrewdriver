# Welcome to Sonic Mint!

This is a customized Linux Mint installation powered by SonicScrewdriver v2.

## Getting Started

1. **Open Terminal** — Press `Ctrl+Alt+T` or click the terminal icon
2. **Type `sonic`** — See all available commands
3. **Create a USB** — `sonic usb create /dev/sdX`
4. **Join the Mesh** — `sonic mesh init`
5. **Enroll Security** — `sonic security enroll`

## What's Included

- **SonicScrewdriver CLI** — Full system toolkit
- **Mesh Networking** — Peer-to-peer with MeshCore
- **CHASIS Games** — Retro gaming library
- **Security Tools** — FIDO2, GPG, SSH key management
- **Device Library** — Hardware database with repurposing guides

## Need Help?

- `sonic --help` — CLI reference
- `sonic usb --help` — USB commands
- `sonic mesh --help` — Mesh networking
- `sonic security --help` — Security enrollment

Enjoy your Sonic experience!
